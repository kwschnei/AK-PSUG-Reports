Attendance Description

Enrollment Description
New Student List The report lists students who have enrolled after school started.
Transfer Students Report This report lists all students who have transferred out of school this year.

Grades Description
Gradebooks Not Updated Report (Java grade book) The report lists teachers who have not updated their Java grade book in the last week. This report will not work for sections using the PowerTeacher Pro grade book.
Gradebooks Not Updated Report PTP The report lists teachers who have not updated their PowerTeacher Pro Gradebook in the last week. This report will not work for sections using the Java grade book.
Gradebooks Not Updated Report (Java grade book), Student List The report lists the students who have not had their grades updated in the last week (in the Java grade book). This report will not work for sections using the PowerTeacher Pro grade book.
Gradebooks Not Updated Report (PTP), Student List The report lists the students who have not had their grades updated in the last week (in the PowerTeacher Pro grade book). This report will not work for sections using the Java grade book.
Students with Ds and Fs Lists classes in which students have a D or F and attendance totals.
